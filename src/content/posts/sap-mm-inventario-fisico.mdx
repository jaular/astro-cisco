## Crear documento para inventario

Código de transacción `MI01`

El bloqueo de contabilización especifica que durante el recuento de inventario no se podrán contabilizar movimientos de mercancías para los stocks de un material contenidos en el documento para inventario. Para eliminar discrepancias durante el recuento de inventario físico, debe activar el indicador `Bloqueo contabil. / Posting Block`. El bloqueo de contabilización se elimina automáticamente cuando se registran los resultados del recuento del inventario físico.

El indicador `Fijar stock teórico / Freeze book inventory` origina que el stock teórico actual se conserve en el documento para inventario. El sistema compara el stock contado con el stock teórico fijado para comprobar si existen diferencias de inventario.

## Nuevo recuento de inventario (Opcional)

Código de transacción `MI11`

Si la gerencia no acepta la discrepancia, o si la discrepancia supera cierta tolerancia, entonces esos materiales deben volver a contarse. Un recuento permite a los usuarios volver a contar el material en la ubicación en el documento de inventario físico.

## Contabilizar diferencias

Código de transacción `MI07`

Para cada material en el documento de recuento, se debe ingresar la diferencia y un código de motivo para mostrar claramente por qué el recuento no se corresponde con la cantidad del libro.
