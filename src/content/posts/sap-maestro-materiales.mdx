## Material

Productos que serán adquiridos para ser usados en cualquier actividad de la
empresa. Se pueden usar para vender, producir, mantener o consumir.

### Crear material

Código de transacción `MM01`

El `Tipo de material / Material Type` es un atributo básico de todos los materiales que se crean en SAP. Un tipo de material es un agrupador de aquellos materiales que se gestionan del mismo modo.

Los tipos de materiales estándar comúnmente utilizados en **MM** y **PM** son:

| Código | Descripción                         |
| ------ | ----------------------------------- |
| ROH    | Materia prima                       |
| HALB   | Producto semielaborado              |
| FERT   | Producto terminado                  |
| HAWA   | Mercadería                          |
| HIBE   | Materiales auxiliares / Combustible |
| NLAG   | Material no de almacén              |
| ERSA   | Piezas de recambio                  |
| FHMI   | Medios auxiliares de fabricación    |
| IBAU   | Conjuntos de mantenimiento          |

#### Datos base 1

La `Unidad de medida base / Base Unit of Measure` es la unidad de medida más pequeña en que la empresa mantiene su inventario.

El `Grupo de articulos / Material Group` refleja un método para agrupar materiales similares.

![MM01 Datos base 1](https://res.cloudinary.com/jaular/image/upload/v1719430750/blog/sap-materiales/mm01-4_ptkjb6.png "MM01 Datos base 1")

#### Compras

La unidad variable de medida de pedido `UMP var. / Var. OUn` permite que la unidad de medida base de compras se pueda cambiar durante la creación del pedido de compras.
